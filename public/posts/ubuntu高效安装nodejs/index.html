Ubuntu高效安装Nodejs

五月 21, 2024

选择版本：

# 这里安装的是16版本，你可以将16改成其他的版本数字
curl -sL https://deb.nodesource.com/setup_16.x -o nodesource_setup.sh

运行配置脚本

sudo bash nodesource_setup.sh

安装

sudo apt install nodejs

然后运行 node -v，搞定。
